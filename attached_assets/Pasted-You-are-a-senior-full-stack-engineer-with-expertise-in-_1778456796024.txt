## CORE FEATURES

### 1. CIFRAS (Chord Charts)
- Create cifra with fields: title, artist, key (tom), lyrics + chords inline (plain text editor)
- Import cifra from .txt file (file input button)
- Edit cifra (title, artist, key, full content)
- Delete cifra (with confirmation modal)
- View cifra in full-screen reading mode with large, legible font

### 2. TRANSPOSIÇÃO DE TOM (Transpose)
- In the cifra view, show current key (tom) with two buttons: ▲ (subir) and ▼ (baixar)
- Each press shifts all chords in the content up or down by one semitone
- Support all standard chord notations: C, D, E, F, G, A, B and their sharps/flats (#/b)
- Also support Portuguese notation if present (Dó, Ré, Mi, etc.) — map to standard
- Display current key prominently during transposition

### 3. HINÁRIOS (Songbooks)
- Create named hinários (e.g., "Hinário 1 – Louvor", "Hinário Cristã")
- Add existing cifras to a hinário (multi-select from cifra list)
- Remove cifras from a hinário
- Delete hinário (with confirmation)
- View hinário — shows list of its cifras in order; tap to open cifra

### 4. BUSCA (Search)
- Global search bar in header/top bar
- Searches across cifra titles and artists in real time (debounced, 300ms)
- Also searches hinário names
- Results grouped by type (Cifras / Hinários) with icons

### 5. ROLAGEM AUTOMÁTICA (Auto-Scroll)
- Available in full-screen cifra reading mode
- Button to start/pause auto-scroll
- Speed slider: slow / medium / fast (3 presets)
- Auto-scroll pauses when user touches/scrolls manually; resumes after 2s

### 6. PERFIL & BACKUP
- Profile screen: user sets their name and profile photo (stored locally)
- Backup: serialize all app data (cifras + hinários) to a JSON string
- Generate a shareable link using base64 encoding of the JSON: 
  format → https://[app-url]/import?data=[base64]
- Import: if URL contains ?data= param on load, decode and merge data into local state
  (show confirmation modal before importing: "Deseja importar X cifras e Y hinários?")
- Also offer raw JSON download as .json file

## DATA MODELS (TypeScript)

interface Cifra {
  id: string;           // uuid
  title: string;
  artist: string;
  key: string;          // e.g. "C", "G#", "Bb"
  content: string;      // full chord+lyric text
  createdAt: number;
  updatedAt: number;
}

interface Hinario {
  id: string;
  name: string;
  cifraIds: string[];
  createdAt: number;
}

interface UserProfile {
  name: string;
  photoUrl: string | null;   // base64 data URL
}

## TRANSPOSITION LOGIC (transpose.ts)

- Define chromatic scale: ["C","C#","D","D#","E","F","F#","G","G#","A","A#","B"]
- Define flat equivalents map: {"Db":"C#", "Eb":"D#", "Gb":"F#", "Ab":"G#", "Bb":"A#"}
- Parse content text: find chord tokens (words matching /^[A-G][#b]?(m|maj|min|dim|aug|sus|add|\d)*/)
- Shift each chord root by +1 or -1 semitone on the chromatic scale
- Preserve chord quality (m, maj7, sus4, etc.)
- Return transformed content string
