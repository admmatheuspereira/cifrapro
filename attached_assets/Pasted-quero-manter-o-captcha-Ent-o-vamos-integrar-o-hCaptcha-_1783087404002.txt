Integrar hCaptcha no formulário de login/cadastro.

O Supabase usa hCaptcha (não reCAPTCHA). Siga esses passos:

Instale a dependência:
npm install @hcaptcha/react-hcaptcha
No arquivo src/pages/Auth.tsx, adicione:
tsx
import HCaptcha from '@hcaptcha/react-hcaptcha'
import { useRef } from 'react'

// Dentro do componente, adicione:
const captchaRef = useRef<HCaptcha>(null)
const [captchaToken, setCaptchaToken] = useState<string | null>(null)

// Adicione o componente antes do botão Entrar:
<HCaptcha
  sitekey="SEU_SITEKEY_AQUI"
  onVerify={(token) => setCaptchaToken(token)}
  onExpire={() => setCaptchaToken(null)}
  ref={captchaRef}
  theme="dark"
/>

// No handleLogin e handleCadastro, passe o token ao Supabase:
const { error } = await supabase.auth.signInWithPassword({
  email,
  password,
  options: { captchaToken: captchaToken ?? undefined }
})

// Após login/cadastro, reseta o captcha:
captchaRef.current?.resetCaptcha()
setCaptchaToken(null)

Depois manda esse prompt no Replit:

Integrar hCaptcha no formulário de login/cadastro.

1. Instale:

npm install @hcaptcha/react-hcaptcha
2. No arquivo src/pages/Auth.tsx, faça as seguintes alterações:

tsx
// Adicione nos imports:
import HCaptcha from '@hcaptcha/react-hcaptcha'
import { useRef } from 'react'

// Dentro do componente, adicione esses estados:
const captchaRef = useRef<HCaptcha>(null)
const [captchaToken, setCaptchaToken] = useState<string | null>(null)

// Adicione o componente hCaptcha antes do botão "Entrar":
<HCaptcha
  sitekey="ea2cf4e2-a202-4776-a410-7914672c6c98"
  onVerify={(token) => setCaptchaToken(token)}
  onExpire={() => setCaptchaToken(null)}
  ref={captchaRef}
  theme="dark"
  size="invisible"
/>

// No handleLogin, passe o token:
const { error } = await supabase.auth.signInWithPassword({
  email,
  password,
  options: { captchaToken: captchaToken ?? undefined }
})
captchaRef.current?.resetCaptcha()
setCaptchaToken(null)

// No handleCadastro, passe o token:
const { error } = await supabase.auth.signUp({
  email,
  password,
  options: { captchaToken: captchaToken ?? undefined }
})
captchaRef.current?.resetCaptcha()
setCaptchaToken(null)
Como o modo é 99.9% Passive, o captcha é invisível — não aparece nenhum widget visual para o usuário, só valida em background.
